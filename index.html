Respostas ao Questionário de Requisitos

Analisamos cada requisito e demonstramos como nosso ecossistema Big4Tech oferece as ferramentas necessárias para construir uma solução que vai além do e-commerce tradicional.

Requisitos do Projeto

Solução Técnica:

A integração é viável e será tratada como um projeto à parte, utilizando nossa plataforma de integração Integra.do para conectar a loja ao ERP.

Solução Técnica:

A integração é realizada através de uma ferramenta parceira, a Gubee. O plano é definido por volume de produtos.

  • Envio de status via WhatsApp: Viável através da Integra.do, porém requer a contratação de uma solução de disparador de WhatsApp.
  • Cashback / Bônus: A loja possui um módulo de clube de fidelidade nativo. A comunicação via WhatsApp não está inclusa, sendo sugerido o uso de ferramentas externas como a CRM Bônus para esta finalidade.
  • Promoções Exclusivas: Funcionalidade padrão da plataforma.

Solução Técnica:

A plataforma Bis2Bis possui integração padrão com a Frenet e outras plataformas de frete.

Solução Técnica:

A plataforma gera o feed de produtos necessário. A configuração na plataforma Meta deve ser realizada pela equipe de marketing do cliente.

Solução Técnica:

  • A plataforma possui um módulo de afiliados nativo para gestão de cupons e links de parceiros.
  • Solução Ideal: Para uma operação completa, incluindo pagamento de comissões, a solução recomendada é a Flip.net, nossa plataforma de Social Selling.

Solução Técnica:

A plataforma possui o módulo nativo de "Produto Pacote", que permite ao cliente selecionar as variações dos itens dentro do kit.

Requisitos Gerais e Institucionais

Solução Técnica:

Nosso time de design criará uma página institucional customizada com as fotos e vídeos fornecidos.

Solução Técnica:

A incorporação do mapa é um módulo adicional que pode impactar a performance. A sugestão é utilizar um link para o Google Maps no rodapé ou na página de contato.

Solução Técnica:

São funcionalidades padrão da plataforma, a serem inseridas no rodapé e na página de contato.

Solução Técnica:

Será desenvolvido um módulo de formulário customizado para B2B para captar os dados dos interessados.

Solução Técnica:

É uma customização para criar uma página específica para o catálogo digital com um botão de download.
